# Mac Pro 2019 Local AI Guides

This repository contains my guides, scripts, patches, notes, and experiments for setting up the Mac Pro 2019 as a local AI workstation.

The focus is on Ubuntu, ROCm, AMD Radeon PRO MPX GPUs, PyTorch, Triton, vLLM, Ollama, Infinity Fabric Link / xGMI experiments, Duo MPX GPU BAR handling, and related infrastructure needed for local AI workloads.

These guides are based on my own tested Mac Pro 2019 systems, including dual AMD Radeon PRO W6900X MPX modules and dual AMD Radeon PRO W6800X Duo MPX modules. Everything here is experimental, community-oriented, and provided as-is.
